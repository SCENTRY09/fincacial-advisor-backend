TITLE: Banking Safety and Account Security

CATEGORY: RBI/Banking

SUMMARY: Banking safety involves protecting account information and following security practices. RBI guidelines ensure customer protection and fraud prevention.

KEY POINTS:
- Protect account credentials
- Use strong passwords
- Enable two-factor authentication
- Monitor account regularly
- Report suspicious activity
- Use secure networks

SECURITY PRACTICES:
- Never share OTP or PIN
- Don't use public WiFi for banking
- Update passwords regularly
- Use unique passwords
- Enable biometric authentication
- Verify URLs before login

ACCOUNT PROTECTION:
- FDIC insurance: Up to Rs. 5,00,000
- Deposit insurance: Covers deposits
- Fraud protection: RBI guidelines
- Dispute resolution: Customer protection
- Grievance redressal: Bank ombudsman
- Compensation: For unauthorized transactions

BEST SUITED FOR:
- All bank account holders
- Online banking users
- Mobile banking users
- Credit card users
- Loan borrowers
- Anyone with bank account

RISK LEVEL: Low (with precautions)

IMPORTANT GUIDELINES:
- Keep credentials confidential
- Use strong passwords
- Enable security features
- Monitor transactions
- Report fraud immediately
- Update contact details

COMMON MISTAKES:
- Sharing OTP or PIN
- Using weak passwords
- Banking on public WiFi
- Ignoring suspicious activity
- Not updating passwords
- Clicking phishing links

SOURCE TYPE: RBI Guidelines

FRAUD PREVENTION:
- Verify sender before responding
- Don't click suspicious links
- Check URL before login
- Verify caller identity
- Report phishing attempts
- Use official app only

ACCOUNT MONITORING:
- Check balance regularly
- Review transactions
- Set up alerts
- Verify statements
- Report discrepancies
- Keep records

FRAUD RESPONSE:
1. Stop using account
2. Contact bank immediately
3. File complaint
4. Provide documentation
5. Follow bank instructions
6. Monitor account closely
